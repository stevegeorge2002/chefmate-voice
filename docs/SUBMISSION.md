\# ChefMate Voice - Hackathon Submission



\## Project Name

\*\*ChefMate Voice\*\* - Multilingual Kitchen Voice Command System



\## Team

\- Steve (Lead Developer) - Northeastern University



\## Tagline

Breaking language barriers in professional kitchens with real-time voice translation



\## Problem

Restaurant kitchens face critical communication challenges: dirty hands can't touch screens, staff speak different languages, manual paperwork wastes time, allergen mistakes risk lives.



\## Solution

Voice-activated system with real-time translation across 5 languages, automatic allergen warnings, smart timing coordination, and hands-free operation.



\## Technology

\- \*\*Speechmatics:\*\* Real-time speech-to-text with noise/accent handling

\- \*\*MiniMax:\*\* Multilingual AI for command parsing and translation

\- \*\*ArmorIQ:\*\* Food safety validation and compliance
