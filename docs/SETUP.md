\# ⚡ ChefMate Voice - Quick Setup



\## For Restaurant Testing



\### 1. Clone Repository

```bash

git clone https://github.com/stevegeorge2002/chefmate-voice.git

cd chefmate-voice/backend

```



\### 2. Install

```bash

pip install -r requirements.txt

```



\### 3. Run

```bash

python kitchen\_agent.py

```



\### 4. Open Dashboard

Browser: http://localhost:8000



\### 5. Test Voice Commands



Click voice button and say:

\- "Order fire table 5, two steaks medium-rare"

\- "How long on table 3?"

\- "86 salmon"

\- "Table 5 plated"
